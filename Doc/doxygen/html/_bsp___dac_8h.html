                    ##### How to use this driver #####
===============================================================================
  1,    适用芯片
        STM      :  STM32F1  STM32F4
        Nordic   :  Nrf51    Nrf52
  2,    移植步骤
  3,    验证方法
  4,    使用方法
  5,    其他说明
             DAC特性:
             1、两路
             2、12bit时数据可被配置成左对齐或右对齐
             3、具有同步更新能力
             4、噪声产生
             5、三角波产生
             6、两个通道独立转换或同步转换
             7、每个通道都有DMA能力
             8、DMA出错检测
             9、可外部触发
             可以设置成定时器触发，TIM6和TIM7 是专公DAC转换用的定时器。初始化也很方便。
