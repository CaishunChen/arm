===============================================================================
                    ##### How to use this driver #####
===============================================================================
  1,    适用芯片
        STM      :  STM32F1  STM32F4
        Nordic   :  Nrf51    Nrf52
  2,    移植步骤
        2.1   在Bsp_Led.h中设定时钟与引脚信息。
        2.2   在主函数调用            BspLed_Init()
        2.3   在100mS循环函数中调用   BspLed_100ms()
  3,    验证方法
        3.1   在Debug环境加入调试指令，执行后所有LED循环点亮，退出后所有LED关闭
  4,    使用方法
        4.1   用户可以调用Bsp_Led.h中的宏定义直接控制LED
  5,    其他说明
